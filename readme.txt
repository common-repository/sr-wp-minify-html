SR Minify HTML helps you reduce your front end page size by removing unwanted newlines, extra spaces, html, css and javascript comments. It basically minifies the rendered html and delivers only one line or effective html code.

== Description ==
SR Minify HTML compresses the HTML output and removes all new lines, extra spaces, HTML comments, CSS and JS comments single and multiline both and delivers only one single line of HTML code to the client side. This will help you reduce the page size and apparently the load time.
This plugin will help you increase your SEO score.

== Installation ==
1. Upload the entire sr-minify-html folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. You will find ‘SR MINIFY HTML’ as a sub-menu under your WordPress Settings menu.

== Frequently Asked Questions ==
==This plugin will work perfectly if your theme is created in the standard way.==
1. Target template should start with get_header() before outputting.
2. Your theme footer or end of file should have wp_footer() just before closing body tag.
